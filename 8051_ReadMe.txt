Before compile the code, please make sure your computer has installed the sdcc compiling environment.
Please follow the rules below to compile and load the code:
1. Open the "code" folder, and enter "cmd" in the path bar to invoke the commend line.
2. Enter "sdcc GateC.c" in commend line to compile the code.
3. Enter "packihx GateC.ihx > GateC.hex" in commed line to generate GateC.hex file.
4. Then choose the GateC.hex file to load to the 8051 microprocessor.
5. Open the switch of 8051 and wait for a second then the program is loaded.
